Asciinema-rails Demo

Asciinema-rails is a gem that allows you to generate playback files from Asciinema and Sudosh logs and host them from your own website using the bundled player.

Host Your Own Asciinema Asciicasts

The asciicast below is an example of a recording made with the Asciinema recorder that was converted to a playback format using the asciinema-rails gem.

Asciinema allows you to record asciicasts for uploading to asciinema.org. Asciinema-rails extracts the asciinema.org conversion classes and bundles them in a standalone gem that also contains the asciinema.org player components. This allows you to create your own playback files and play them from your own website or app.

The gem also provides a utility for converting Sudosh log files to the asiinema format - like Asciinema, Sudosh records all keystrokes and output to a logfile, but it is an auditing shell filter and can be used as a login shell (i.e. it automatically records sessions afer login).
